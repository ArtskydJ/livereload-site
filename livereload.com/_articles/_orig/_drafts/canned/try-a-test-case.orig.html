2. Add the folder to LiveReload, enable compilation
3. Save style.less once to get it compiled
4. Uncomment the SCRIPT line at the end of index.html if you're testing with file: URL in Safari (or, alternatively, host the files on your local web server and use the browser extension)
5. Open index.html in your browser
6. Make some changes to index.html and style.less, check if they get applied.
