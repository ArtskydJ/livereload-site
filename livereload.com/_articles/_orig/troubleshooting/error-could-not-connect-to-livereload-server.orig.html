This error indicates that the LiveReload extension cannot connect to the LiveReload application. Here are a few things to check:

  • is the LiveReload app running?
  • do you have the latest version of LiveReload app?
  • are you running third-party firewall software that might have blocked LiveReload?
As of Mac version 2.3.5 and Windows version 0.0.4, no known reasons for this problem exist. Please contact us if you're getting the message!
